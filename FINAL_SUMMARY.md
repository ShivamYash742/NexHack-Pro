## 🎯 How to Update AI Prompts

### Quick Method:
1. Open `lib/prompts.json`
2. Find the prompt you want to change
3. Edit the text (keep `{variables}` intact)
4. Save and restart server

### Example:
```json
{
  "resumeSummary": {
    "prompt": "Analyze this resume and provide a 2-3 sentence summary: {resumeText}"
  }
}
```
